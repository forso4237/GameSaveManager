Game Save Manager v1.0.0

- New: Named saves — saving now opens a custom window that asks you to name the save and pick a folder. The name is pre-filled with something sensible (e.g. "Logs R34"). Re-using a name in the same folder overwrites it.
- New: Folders — organize saves into folders (default "General"). Create folders with the + button and switch with the < > buttons in the catalog or the on-screen panel. Old loose saves appear under a "Legacy" folder.
- New: Share codes — generate one block of text (GSM1...) that holds an ENTIRE save (towers, positions, upgrades, cash, lives). Copy it from a save's "Code" button and add it via "Import code" (paste or from clipboard).
- New: File export/import — export a save to the _Exports folder, or drop .json files into _Imports and pull them in with "Import files".
- Changed: The on-screen panel is now HIDDEN by default. Use F6/F7 or the mod menu, or set "Button location" to show the panel on-screen or during pause.
- Changed: F7 now quick-loads your most recent save (across all folders) instead of a fixed slot.
